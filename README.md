Wicked Fabrication Portfolio
Overview

Wicked Fabrication is a custom web application combining React for the frontend and Drupal for content management on the backend. This project showcases a dynamic portfolio and gallery system with responsive design, filtering, and modal interactions, all powered by modern web technologies.

The project is a work-in-progress with a focus on:

Integrating a React frontend with a Drupal backend.

Clean, modular, and responsive component design.

Handling media and portfolio items dynamically.

Features

React-based portfolio gallery: Sections for different project categories with filterable tags.

Modal image viewer: Click on portfolio items to view larger versions in a responsive modal.

Drupal content integration (in progress): Preparing fetch requests to dynamically load portfolio items from Drupal.

Clean codebase and structure: Uses TailwindCSS for styling and keeps backend files (Drupal runtime, caches) ignored in Git.

Tech Stack

Frontend: React, TailwindCSS, JavaScript

Backend: Drupal CMS

Other tools: Git, GitHub
